# StrongSalon (FitLog v3.0)

모바일 퍼스트 피트니스 기록·통계·랭킹 웹앱.

## 실행

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 접속.

## 문서

- **[전체 아키텍처](docs/README.md)** — 프로젝트 구조, 탭 시스템, 컴포넌트, 데이터 흐름
- **[기획 수정안 (최종본)](docs/PLANNING.md)** — 탭 변경, 목표 추적기, 당근 크레딧, 운동사전
- **[백엔드 설계](docs/BACKEND_README.md)** — NestJS 엔티티·API (예정)
